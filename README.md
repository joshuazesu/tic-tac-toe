# ❌ TicTacToe ⭕

The classic TicTacToe game built with React and Javascript.
This is my first GitHub project - just a fun way to learn and practice.

## 👨🏼‍💻 Technologies
- `React`
- `JavaScript`
- `HTML`
- `CSS`
- `Firebase`

## 🎮 Features
- **Two-player mode** — *take turns placing X and O*
- **Win detection** — *automatically detects when someone wins*
- **Draw detection** — *knows when it's a tie*
- **Reset button** — *start a new game anytime*

## 🚀 Getting Started
To run this project locally:

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm start`
4. Open `http://localhost:3000` in your browser
